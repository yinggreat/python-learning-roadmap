# 转行项目型 - Python 12 周学习计划

- 学习画像：适合希望通过作品集和项目案例转向 Python 岗位的人。
- 每周投入：10 小时
- 特别关注：automation
- 强调能力：项目作品 / 代码规范 / 面试表达

## Weekly Plan

- 本周目标：Build a clean mental model of Python syntax, data structures, functions, modules, files, and exceptions.
- 重点关键词：variables
- 建议项目：command line todo app

## 建议执行方式
- 每周至少保留 1 次复盘，把遇到的报错和解决办法记录下来。
- 每 2 周输出一个小项目，而不是只记语法笔记。
- 第 12 周把最好的项目整理成 GitHub 作品集入口。
